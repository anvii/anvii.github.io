Base demo

This is a collection of Vue3 components designed to create menus in web applications. The collection consists of:

  • GMenu
  • GMenuItem
  • GMenuBar
  • GContextMenu

See examples below

  • Open
  • Open recent

    • Project 3
  • Save
  • Close (disabled)

Context menu

Right-click here to open context menu 
<template>
  <div class="showbox show-contextmenu" @contextmenu.prevent="$refs.menu.show($event)">
    Right-click here to open context menu
    <g-context-menu ref="menu" autohide>
      <g-menu-item label="New" close-menu />
      <g-menu-item label="Open" close-menu />
